🍎 دليل رفع التطبيق على iOS

بدون Mac - سهل وسريع!

🎉 تم الإعداد بنجاح!
المشروع جاهز للرفع على App Store باستخدام Codemagic أو GitHub Actions
1 📝 التسجيل في Apple Developer

أول خطوة هي التسجيل في برنامج مطوري Apple:

🔗 افتح Apple Developer
⚠️ ملاحظة: التكلفة 99 دولار سنوياً
  • احتفظ بـ Apple ID الخاص بك
  • استخدم بطاقة ائتمان صالحة
  • انتظر 24-48 ساعة للموافقة
2 🚀 التسجيل في Codemagic

Codemagic سيبني التطبيق تلقائياً على سيرفرات Mac في السحابة:

🔗 سجل في Codemagic (مجاناً)
1. Sign up with GitHub
اضغط على زر GitHub واسمح بالوصول
2. Add application
اختر مشروع test123 من القائمة
3. Select Flutter
اختر Flutter كنوع المشروع
4. Finish setup
Codemagic سيكتشف codemagic.yaml تلقائياً!
3 ⚙️ إعداد App Store Connect

أنشئ التطبيق في App Store Connect:

🔗 افتح App Store Connect
My Apps → ➕ New App Platform: iOS Name: أولديزز وركرز Bundle ID: com.oldies.attendance SKU: oldies-workers-001 Language: Arabic
4 🔗 ربط Codemagic مع Apple

في Codemagic Dashboard:

Teams → Your team → Integrations → App Store Connect → Enable → Connect with Apple ID
✨ سحر! Codemagic سيعمل certificates و provisioning profiles تلقائياً!
5 📧 تحديث البريد الإلكتروني

افتح ملف codemagic.yaml وغيّر:

email: recipients: - your-email@example.com # 👈 ضع بريدك هنا
6 🎉 ابدأ البناء!

في Codemagic Dashboard:

Applications → test123 → Start new build → Select Workflow: ios-workflow → Start new build
⏱️ انتظر 15-20 دقيقة... استرح واشرب قهوة! ☕
🎊 تم! بعد اكتمال البناء، التطبيق سيُرفع تلقائياً على TestFlight!
7 📱 التجربة على TestFlight

جرب التطبيق قبل النشر:

🔗 افتح TestFlight
  • أضف مستخدمين للتجربة (حتى 100 مستخدم داخلي)
  • أرسل لهم دعوة عبر البريد
  • حمّل تطبيق TestFlight من App Store
  • جرب التطبيق وأصلح الأخطاء
8 🌟 النشر على App Store

جاهز للنشر؟ في App Store Connect:

My Apps → أولديزز وركرز → Prepare for Submission → املأ جميع البيانات: • Screenshots • Description • Keywords • Privacy Policy → Submit for Review
🎯 النجاح! انتظر 24-48 ساعة لمراجعة Apple ثم التطبيق سيكون متاح على App Store! 🎉

🆘 مشاكل شائعة

  • Build Failed: تحقق من الـ logs في Codemagic
  • Certificate issues: تأكد من ربط App Store Connect
  • Missing screenshots: استخدم أداة مثل Figma لإنشائها
